## What is Terraform?
Terraform is an Infrastructure as Code (IaC) tool developed by HashiCorp.
It allows infrastructure resources to be defined, provisioned, and managed
using declarative configuration files.

---

## Why Use Terraform?
- Automates infrastructure provisioning
- Ensures consistency and repeatability
- Easy to version control
- Cloud-provider independent

---

## What is AWS?
Amazon Web Services (AWS) is a cloud computing platform that provides
on-demand compute, storage, networking, and many other services.

Key AWS services used:
- EC2 (Elastic Compute Cloud)
- IAM (Identity and Access Management)

---

## How Terraform Works with AWS
Terraform uses AWS credentials configured via AWS CLI.
It communicates with AWS APIs to create and manage resources.

---

## Basic Terraform commands

1) terraform init
2) terraform plan
3) terraform apply
4) terraform destroy
